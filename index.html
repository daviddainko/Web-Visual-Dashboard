Summary: Latitude vs. X

The purpose of this project was to analyse how weather changes as one gets closer to the equator. To accomplish this analysis, we first pulled data from OpenWeatherMap API to assemble a dataset on over 500 cities

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This site provides the source date and visualizations created as part of the analysis, as well as explainations and descriptions of any trends and correlations witnessed.
